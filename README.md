# David Bakery Admin

Administrative system for managing bakery orders, customers, payments, inventory, and finances.

## Project Overview

David Bakery Admin is a web-based administrative system designed for a bakery business.  
The system will help manage customer orders, delivery dates, payments, inventory, expenses, income, and internal users.

The goal of this project is to provide a simple, organized, and scalable platform that improves the daily management of the bakery.

## Main Features

- User authentication and role-based access
- Customer management
- Order registration and tracking
- Delivery date control
- Payment tracking
- Receipt or ticket generation
- Inventory management
- Income and expense management
- Basic administrative dashboard

## Technologies

- C#
- ASP.NET Core
- Razor Pages
- Web API
- SQL Server
- JWT Authentication
- GitHub Actions
- Azure App Service
